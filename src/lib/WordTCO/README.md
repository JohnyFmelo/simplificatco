# WordTCO

Pasta reservada para a montagem do arquivo do TCO em formato Word (DOCX).

Objetivo:
- Centralizar utilitários e funções para gerar o documento do TCO em DOCX.
- Substituir completamente o fluxo anterior baseado em PDF.

Próximos passos (sugestão):
- Definir o schema dos dados do TCO que serão inseridos no documento.
- Implementar um gerador usando uma biblioteca como `docx` para Web.
- Adicionar função de download/salvamento do `.docx` e integração com Storage se necessário.
